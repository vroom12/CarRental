## 先进入服务端启动服务器

```sh
cd car-rental-server
```

# 注入依赖

## npm

```sh
npm install
```

## yarn

```sh
yarn
```

# 开启服务器

## dev

```sh
nest start -w
```

## pro

```sh
nest start:pro
```

## 开启服务器后进入 UI 页面

```sh
cd .. && cd CarRental-UI
```

# 注入依赖

## npm

```sh
npm i
```

## yarn

```sh
yarn
```

# 运行

## 开发环境

### npm

```sh
npm run dev
```

### yarn

```sh
yarn dev
```
